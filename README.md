Απλό project To do List φτιαγμένο με JavaScript

## Τεχνολογίες
- HTML
- CSS
- JavaScript

## Χρήση
1. Άνοιξε το `index.html`.
2. Γράψε το task στο input box.
3. Πάτησε enter ή το κουμπί για να το προσθέσεις.
4. Κάνε κλικ στο task για να το μαρκάρεις ως ολοκληρωμένο.
5. Κάνε κλικ στο × για να το διαγράψεις.
